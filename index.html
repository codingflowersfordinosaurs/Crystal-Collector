Crystals Collector

You will be given a random number at the start of the game

There are four elements below. By clicking on an element you will add a specific amount of points to your total score

You win the game by matching your total score to the random number, you lose the game if your total score goes above the random number

The value of each element is hidden from you until you click on it

Each time when the game starts, the game will change the values of each element
